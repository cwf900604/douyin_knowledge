# [SOP] 询盘处理标准流程

> 目标：所有询盘 24 小时内首次回复，72 小时内完成报价，7 天内样品/试单确认

---

## 一、询盘分级 (Inquiry Triage)

| 级别 | 标志 | 响应时间 | 策略 |
|------|------|----------|------|
| 🔥 S级 | 明确说出"想下单""发PI"、提供详细规格需求 | <4小时 | 立即回复，直接发PI/报价 |
| ⭐ A级 | 询问价格、MOQ、技术参数 → 采购意向明确 | <12小时 | 标准报价模板 + 产品目录 |
| 📋 B级 | 泛泛咨询"有什么产品""价格多少" | <24小时 | 发目录 + 提问引导明确需求 |
| 🗑️ C级 | "你好""在吗"无下文 / 明显的钓鱼/骗子 | 无回复/模板拒绝 | 不浪费精力 |

---

## 二、S级询盘快速响应流程 (<4h)

```
1. 收到询盘 → 立即判断是否S级
2. 确认库存/交期 → 查工厂排期
3. 计算运费 → 海运/空运/快递 3 选报价
4. 做 PI (Proforma Invoice) → 模板填充
5. 发送 + WhatsApp 通知客户
```

**S级回复模板要点**：
- 感谢询盘，确认需求
- 附上 PI + 产品规格书
- 提供 2-3 个物流选项（空运快/海运省）
- 明确付款方式 (T/T 30% 预付 + 70% 发货前 / L/C / 阿里信保)
- 强调交期和售后保障

---

## 三、A 级询盘标准回复 (<12h)

```
必含内容:
☑ 公司简介 (1-2句话)
☑ 产品规格表 (针对客户询的产品)
☑ 价格区间 (根据数量分层)
☑ MOQ说明
☑ 样品政策 (样品费可退 if 下大货)
☑ 认证清单
☑ 交期预估
☑ WhatsApp/微信联系方式
```

---

## 四、报价分层策略

| 采购量 | 定价策略 | 利润率目标 |
|--------|----------|------------|
| 样品单 (1-10pcs) | 零售价，不加利润 | 覆盖成本即可 |
| 小批量 (10-100pcs) | 出厂价 × 1.3 | 20-25% |
| 中批量 (100-1000pcs) | 出厂价 × 1.15 | 10-15% |
| 大批量 (1000+pcs) | 出厂价 × 1.05-1.10 | 5-8%（走量） |
| 年单/长期合同 | 阶梯价（季度调整） | 3-5% + 锁定客户 |

## 五、常见客户疑问标准应答库 (FAQ)

### "你的价格比XX贵"
> 我们的价格反映了 [全金属齿轮 / 日制电位器 / 100%出厂测试] 的品质。如果您的目标价位是 $[X]，我们可以看看简化版 [替代型号] 是否能满足需求。

### "MOQ太高了"
> 首次合作我们理解。可以先试单 [最小量]，单价按 [试单价] 计算。合作稳定后享受批量价。

### "能发免费样品吗"
> 我们提供付费样品（$[X]/件），但如果后续大货订单超过 [$Y]，样品费可全额抵扣大货款。运费由贵司承担。

### "你们的质保政策？"
> 标准质保 [12] 个月。质量问题免费换新。人为损坏提供成本价替换。

### "多久能交货？"
> 常规型号 [X] 个工作日。定制型号 [Y] 个工作日。旺季 [Z] 月需提前下单。

### "接受什么付款方式？"
> T/T (30% deposit, 70% before shipment) / L/C at sight / Alibaba Trade Assurance / PayPal (样品单 <$500)

## 六、样品寄送 SOP

```
Step 1: 确认样品需求 → 发 PI (Proforma Invoice)
Step 2: 收到样品费 → 安排生产/备货
Step 3: 发货 → 拍照/视频给客户 + 物流单号
Step 4: 追踪 → 到达前1天提醒
Step 5: 到达后3天 → 询问测试反馈
Step 6: 收到反馈 → 报价大批量价格
Step 7: 无反馈 → 7天后再次跟进 → 14天后最后跟进 → 转入休眠
```

---

## 七、客户沟通禁忌

| ❌ 不要 | ✅ 应该 |
|--------|--------|
| 报完价就消失 | 报价后 2 天跟进确认收到 |
| 一上来就报最低价 | 留谈判空间，大单才放底价 |
| 对不同客户报不同价 | 统一报价体系，差异化体现在赠品/服务 |
| 贬低竞品 | 客观对比，强调自己的优势 |
| 催客户下单 | 提供价值（如市场趋势、选品建议）而非催促 |
| 用模板回复所有人 | 每个客户至少个性化 1-2 句话 |
