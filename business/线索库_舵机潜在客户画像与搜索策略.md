## 一、客户画像 (Buyer Persona)

### Persona A：RC 模型批发商
| 维度 | 描述 |
|------|------|
| 身份 | RC Hobby Shop 老板 / 采购经理 |
| 规模 | 中小型批发商，年采购量 500-5000 台 |
| 地理 | 美国、德国、日本（RC文化盛行地区） |
| 痛点 | 品牌舵机(Futaba/Savox)进价高、利润薄 → 找高性价比替代 |
| 决策因素 | 价格(40%) > 质量稳定性(30%) > 交期(20%) > 品牌/外观(10%) |
| 采购周期 | 季度采购，旺季前 2 个月下单 |
| 搜索关键词 | `servo motor wholesale` `RC servo bulk` `cheap servo supplier` |

### Persona B：无人机/机器人配件经销商
| 维度 | 描述 |
|------|------|
| 身份 | Drone parts distributor / Robotics component reseller |
| 规模 | 中型经销商，年采购量 1000-10000 台 |
| 地理 | 全球（美国、欧洲、中东无人机产业区） |
| 痛点 | 需要高精度、快速响应舵机，且需稳定供应 |
| 决策因素 | 技术参数(40%) > 稳定性(25%) > 认证(20%) > 价格(15%) |
| 采购周期 | 项目制采购，随客户订单波动 |
| 搜索关键词 | `brushless servo motor` `digital servo high torque` `drone servo supplier` |

### Persona C：工业自动化集成商
| 维度 | 描述 |
|------|------|
| 身份 | Industrial automation company / OEM integrator |
| 规模 | 中大型，年采购量不定 |
| 地理 | 德国、日本、美国、中国（转出口） |
| 痛点 | 定制化需求、长期稳定供应、生命周期管理 |
| 决策因素 | 定制能力(30%) > 认证(25%) > 交期(20%) > 技术参数(15%) > 价格(10%) |
| 采购周期 | 长周期，先样品测试→小批量试产→批量采购 |
| 搜索关键词 | `industrial servo motor manufacturer` `custom servo OEM` |

### Persona D：跨境电商卖家
| 维度 | 描述 |
|------|------|
| 身份 | Amazon/eBay/AliExpress 舵机卖家 |
| 规模 | 小批量高频次（50-200 台/次） |
| 地理 | 全球 |
| 痛点 | 需要快速发货、小起订量、有竞争力的出厂价 |
| 决策因素 | 价格(35%) > MOQ灵活性(30%) > 物流速度(25%) > 品质(10%) |
| 搜索关键词 | `servo motor dropship` `servo small MOQ` `servo for Amazon seller` |

## 二、搜索策略矩阵

### LinkedIn 搜索

```
搜索语法:
"procurement manager" AND (servo OR "RC hobby" OR robotics)
"purchasing manager" AND ("remote control" OR drone OR "industrial automation")
"buyer" AND (servo OR actuator OR "motion control")

行业过滤:
- RC & Hobby industry
- Industrial Automation
- Robotics
- Drone / UAV
- Consumer Electronics
```

**每日任务**：
1. 搜索 → 筛选 20 个目标客户
2. AI 生成个性化连接请求（用 Template 1）
3. 记录到 `[CRM] 客户跟进看板.md`

### B2B 平台搜索

| 平台 | 搜索方法 | 目标 |
|------|----------|------|
| Alibaba.com | 搜 "servo motor" → 看 "Buying Requests" 栏目 | 主动询盘的采购商 |
| Global Sources | 同类目 → 看热门产品下的 Inquiry | 有明确采购意向 |
| Made-in-China | 行业分类 → 找国外买家发布的采购需求 | 中小批发商 |
| ThomasNet | 搜 "servo motor distributors" | 美国工业经销商名录 |

### Google 搜索

```
高级搜索运算符:
"servo motor" + "distributor wanted" site:linkedin.com
"servo" + "looking for supplier" site:alibaba.com
"RC servo" + "wholesale inquiry" filetype:pdf
"servo motor" + "import" + "USA" + "contact"
```

| 群组类型 | 搜索关键词 | 作用 |

## 三、线索优先级评分卡

| 评分维度 | 权重 | 高分信号 | 低分信号 |
|----------|------|----------|----------|
| 采购规模 | 30% | 公司 >50人、多个仓库 | 个人、新注册账号 |
| 行业匹配度 | 25% | 核心业务 = RC/机器人/无人机 | 跨界、与舵机无关 |
| 采购意向 | 20% | 主动发询盘、近期求购帖 | 纯浏览、无采购行为 |
| 联系难度 | 15% | 有公开邮箱/LinkedIn活跃 | 无公开联系方式 |
| 区域价值 | 10% | 美/德/日（高利润市场） | 物流困难地区 |

**评分规则**：>80分（A级-立即联系）| 60-80分（B级-本周联系）| <60分（C级-批量联系）
