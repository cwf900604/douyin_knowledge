## 前置条件

### 营业执照

- 官方要求：营业执照注册满**14天**后才能开店
- 实际情况：只要在**国家企业信用信息公示系统**上能查到即可，一般执照下来后 **1-2天**就能开店
- 差距来源：官方14天是保守口径，实操以公示系统可查为准

### 必备工具

- **子鸟浏览器**：用于IP隔离的指纹浏览器
- **静态住宅美国IP**：在子鸟浏览器内购买

## 注册全流程

### 第一步：购买设备与IP

1. 打开子鸟浏览器 → 点击**购买设备**
2. 设备类型选择 **TK专属**
3. IP类型选择 **静态住宅**
4. 地区选择 **美国**（任意州，如加利福尼亚）
5. 时长先选 **1个月**（后续可续费）
6. 提交订单

### 第二步：关键操作——开启自动续费

> **这是整个流程中最重要的一步，直接决定店铺生死。**

在支付页面，**必须勾选自动续费**。原因：

- 忘记续费 → IP被回收 → 新IP变更 → TK判定账号异常 → **封店**
- IP的稳定性是TK风控系统的核心判定维度之一
- 自动续费可随时手动取消，不用担心被长期绑定

### 第三步：添加账号

1. 在账号功能中点击**添加账号**
2. 所属平台选择 **TikTok**
3. **重点**：店铺类型必须选择 **全球跨境**（第三个选项，不是本土店铺）
4. 账号名称随意填写（如"美区一店""美区二店"）
5. TikTok账号和登录密码可先不填，注册后再补
6. 在绑定设备处绑定已购买的IP
7. 点击确定

### 第四步：启动并注册

1. 点击**打开账号**（首次为"启动"）
2. 点击**注册**
3. 按照营业执照上的信息如实填写
4. 提交审核

## 审核时效

| 口径 | 时间 |
|---|---|
| 官方提示 | 1-3个工作日 |
| 实际体验 | 约**40分钟** |

## 核心要点总结

```
营业执照可查（1-2天）
  → 子鸟浏览器购买设备（TK专用 + 静态住宅 + 美国）
    → 开启自动续费（防IP变更封店）
      → 添加账号（全球跨境）
        → 绑定IP → 注册 → 等待审核（实际约40分钟）
```

---

## 风险提示

1. **IP隔离是刚需**：不使用指纹浏览器直接注册 = 极高封店风险
2. **自动续费 = 保险**：不是可选，是必须。IP变了店就没了
3. **店铺类型别选错**：一定要选"全球跨境"，而非本土店铺
4. **信息真实**：按营业执照如实填写，虚假信息会被拒
