## 操作路径

`Win+I` → 应用 → 安装的应用 → 找到目标软件 → 卸载

---

## 绝对不能卸载（卸载会导致系统异常）

| 软件名称 | 原因 |
|---|---|
| **Intel Graphics Software** | 英特尔显卡驱动和控制面板，卸载会影响画面显示 |
| **Microsoft Edge** | 系统自带浏览器，无法卸载 |
| **所有带 C++ 字样的程序** | 系统核心运行库，卸载会导致大量软件无法运行 |

---

## 建议保留（日常高频使用）

| 软件名称 | 保留原因 |
|---|---|
| **截图工具** | 截图 + 录屏，日常刚需 |
| **媒体播放器** | 播放本地视频 |
| **照片** | 查看电脑图片，系统级看图工具 |

---

## 可以卸载（附原因）

### 系统工具类

| 软件名称 | 卸载原因 |
|---|---|
| **电脑管家** | 微软系统优化工具（类似360），占用系统资源，不用可卸 |
| **Microsoft 365 Office** | 付费AI助手，占用大量系统资源 |
| **Microsoft Clipchamp (Clean up)** | 微软视频剪辑软件，功能不如剪映 |
| **Microsoft To Do** | 待办和提醒软件，不用可卸 |
| **Outlook** | 邮件收发工具，不用或使用其他平台即可卸载 |
| **Power Automate** | 微软自动化工具，普通人用不上 |
| **反馈中心** | 向微软反馈意见的渠道，一般用户用不到 |
| **Microsoft Bing** | 无法直接卸载，但可在高级选项中设为"从不"运行并终止 |

### 游戏与娱乐类

| 软件名称 | 卸载原因 |
|---|---|
| **Xbox** | 微软游戏平台，不玩微软游戏即可卸载 |
| **Xbox 6** | 同上 |
| **棋排列游戏** | 不玩游戏即可卸载 |
| **Microsoft Edge 游戏助手** | 不玩网页游戏即可卸载 |

### 云服务与同步类

| 软件名称 | 卸载原因 |
|---|---|
| **Microsoft OneDrive** | 微软云盘，不需要自动备份/文件同步即可卸载 |

### 通讯与远程类

| 软件名称 | 卸载原因 |
|---|---|
| **快速助手** | 临时远程控制桌面，不需要别人远程控制即可卸载 |
| **远程桌面连接** | 永久远程控制桌面，有安全隐患，强烈建议卸载 |

### 媒体与工具类

| 软件名称 | 卸载原因 |
|---|---|
| **Web媒体扩展** | Edge浏览器的视频解码插件，不看特殊视频格式即可卸载 |
| **录音机** | 纯录音功能，不用电脑录音即可卸载（不影响其他软件录音） |
| **相机** | 独立照相/录像软件，无自拍需求即可卸载（不影响视频聊天和直播） |
| **画图** | 自带画图工具，不用电脑做图即可卸载 |
| **便签** | 桌面便利贴/备忘录，用不着即可卸载 |
| **时钟** | 秒表和计时器功能，用不着即可卸载（不影响右下角时间显示） |
| **计算器** | 纯计算功能，不用即可卸载 |
| **记事本** | Windows 11新版文本文档，不习惯新版格式可卸载 |

### 运行时与广告类

| 软件名称 | 卸载原因 |
|---|---|
| **Windows App Runtime** | 微软应用运行框架，不用微软商店即可卸载 |
| **Microsoft 资讯** | 纯粹的广告软件，频繁弹广告，一秒钟都不要犹豫 |
| **天气** | 右下角天气组件本质是广告入口，毫不犹豫彻底卸载 |

---

## 操作优先级建议

1. **第一优先卸载**: 天气、Microsoft资讯（广告软件，零价值）
2. **第二优先卸载**: Xbox系列、游戏、OneDrive（占资源且不常用）
3. **按需卸载**: 其余工具类软件根据个人使用习惯决定

---

## 注意事项

- 远程桌面连接卸载后需要**重启电脑**才能生效
- 部分软件（如Microsoft Bing）不支持直接卸载，只能通过高级选项限制其后台活动
