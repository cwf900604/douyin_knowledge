# raw/ — 原始素材存放区（抖音 + B站）

## 说明
此目录存放抖音 + B站收藏的原始未处理素材，按平台分文件夹：
- `douyin/`：抖音视频口播文稿、字幕（.srt）、收藏清单（.csv）
- `bilibili/`：B站视频简介、字幕、收藏夹导出清单

## 文件命名规范
```
[日期]-[作者]-[主题].md
示例：
  douyin/20260714-机械老王-舵机调试三步法.md
  bilibili/20260714-硬件大牛-PWM信号精讲.md
```

## B站素材获取方式
1. **B站自带字幕**：视频页 → 字幕按钮 → 复制全文
2. **B站 API**：通过 BV 号获取视频简介和字幕
3. **收藏夹导出**：手动整理收藏夹内容 → 导出为 CSV
4. **剪映导出**：下载视频后导入剪映 → 智能字幕 → 导出 SRT/TXT

## 工作流程
1. 新素材放入对应平台子目录（`raw/douyin/` 或 `raw/bilibili/`）
2. 在 Claude Code 中执行 `/ingest raw/平台/文件名.md`
3. AI 自动蒸馏 → 知识卡片存入 `../distill/单篇笔记/`
4. 执行 `/wiki-update` 去重合并 + 刷新全局索引

## 清理策略
- 蒸馏完成后，原始文稿可删除或移入 `raw/douyin/已处理/` 或 `raw/bilibili/已处理/`
- CSV 清单建议保留用于追溯
