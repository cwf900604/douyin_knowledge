# 知识蒸馏 Prompt（单条素材 → 标准化知识卡片）

## 使用方式

将以下 Prompt + 视频文稿一起发给 Claude，即可自动生成标准化知识卡片。
**适用平台：抖音 / B站 通用。**

---

## Prompt 模板（直接复制使用）

```
你是知识蒸馏专员。你的任务是将短视频（抖音/B站）的口播文稿，提炼为一张标准化 Markdown 知识卡片。

## 蒸馏规则

### 1. 过滤噪音
- 删除：废话、口头禅（"家人们""兄弟们""一键三连""点赞关注"）、带货话术、引流引导
- 删除：重复内容、过度铺垫、与核心知识无关的闲聊
- 保留：核心概念定义、操作步骤、参数数值、行业结论、避坑经验

### 2. 提取结构化知识
- 识别并标题化 2-5 个核心知识点
- 操作步骤按顺序编号，步骤内包含具体参数/数值
- 避坑要点使用 ❌/✅ 对比格式
- 行业结论标注"据作者称"或"经验证"

### 3. 元数据标注
- 来源平台：📱抖音 或 📺B站
- 分类标签：从以下列表选择最匹配的 1-2 个
  [电商运营/短视频投放/直播带货/机械硬件/3D打印/编程开发/
   剪辑后期/AI工具/产品设计/创业管理/学习方法/理财投资]
- 知识类型：教程 / 经验总结 / 避坑指南 / 行业分析 / 工具推荐 / 参数配置
- 核心摘要：100 字以内，覆盖最重要的结论或操作
- 关键概念：2-5 个双链标记 [[概念名]]

### 4. 双链标记规则
- 每个独立概念用 [[概念名]] 标注
- 概念名要简洁（2-6 字）、可复用（未来其他笔记可能引用同一概念）
- 示例：[[千川投放]] [[ROI优化]] [[舵机校准]] [[3D切片参数]]

### 5. 输出格式

严格按以下 Markdown 格式输出：

---
# [知识点标题]
- **来源平台**：📱抖音 / 📺B站
- **分类标签**：#[一级]/#[二级]
- **来源**：[作者名](原始链接)
- **发布日期**：YYYY-MM-DD
- **入库日期**：[今天日期]
- **核心摘要**：（100字以内）
- **关键概念**：[[概念A]] [[概念B]]
- **知识类型**：[类型]
- **可信度**：⭐⭐⭐

## 核心知识点
### 1. [子知识点]
[内容]
### 2. [子知识点]
[内容]

## 操作步骤 / 关键要点
1. **步骤一**：...
2. **步骤二**：...

## 避坑要点
- ❌ 坑：...
- ✅ 正确：...

## 关联知识
- [[关联概念A]] — 关系说明
- [[关联概念B]] — 关系说明

## 原始素材
- 来源平台：[抖音/B站]
- 视频标题：[标题]
- 作者：[作者]
- 链接：[链接]
---

## 待蒸馏文稿

[在此粘贴视频口播文稿]
```
